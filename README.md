# Raspico v0.1.0

一个简单的转接板，主要功能：将Rpi Pico和Rpi 4b的供电、串口和SWD管脚对应连接（可使用OpenOCD、gdb和VSCode等编辑器进行较为方便的嵌入式Pico开发），并引出标注两者所有GPIO针脚（便于接线）。
另外引出一对5v GND引脚为风扇供电。

采用深圳嘉立创公司的立创EDA进行设计，并免费打样。
